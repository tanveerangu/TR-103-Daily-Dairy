Week 1: Planning, Setup, and Initial Development (June 11 - June 16, 2024)

In Week 1, we focused on setting up the project structure, planning the eCommerce site, and integrating key components like authentication and database structure.

Day 1: Project Setup and Initial Planning

Date: Tuesday, June 11, 2024

Set up a Git repository for version control. Planned the website structure: home page, product listings, cart, checkout, payment gateway integration, and user account system. Installed Laravel and set up the basic project structure.

Day 2: Database Design and Model Setup

Date: Wednesday, June 12, 2024

Designed the database schema for users, products, orders, and payments. Created migration files and models in Laravel, setting up relationships between models like Product, Order, and User.

Day 3: Authentication and User Registration

Date: Thursday, June 13, 2024

Implemented user authentication with Laravel Breeze or Jetstream. Set up login, registration, and password reset functionalities along with user roles (admin, customer).

Day 4: Building the Product Model and CRUD Operations

Date: Friday, June 14, 2024

Created the Product model and migration. Set up CRUD operations for products, allowing admins to create, update, and delete products. Created views for displaying products to both admins and customers.

Day 5: Frontend Development (Home Page and Product Listing)

Date: Saturday, June 15, 2024

Developed the homepage with a product listing section. Implemented pagination for product listings and styled the pages using Tailwind CSS or Bootstrap.

Day 6: Cart System Implementation

Date: Sunday, June 16, 2024

Created the Cart model and associated routes/controllers. Implemented functionality for users to add, remove, and update products in the cart. Used Laravel sessions to store cart data.

Day 7: Testing and Debugging

Date: Sunday, June 16, 2024

Tested the cart functionality thoroughly. Debugged and fixed any issues that arose during testing.
