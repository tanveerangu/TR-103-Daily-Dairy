Week 2: Advanced Features and Integration (June 17 - June 22, 2024)

In Week 2, we focused on advanced features like integrating the payment gateway (Razorpay), improving the user interface, and adding product details and reviews.

Day 1: Razorpay Payment Gateway Integration

Date: Monday, June 17, 2024

Integrated Razorpay payment gateway to handle secure transactions on the website. This included setting up Razorpay API, creating payment routes, and handling order payment status.

Day 2: Product Details and Variants

Date: Tuesday, June 18, 2024

Extended the Product model to support multiple variants (e.g., size, color) and added detailed product descriptions, images, and pricing information. Updated product listing pages to display additional information.

Day 3: User Profiles and Order History

Date: Wednesday, June 19, 2024

Developed a user profile system where customers can view their order history, edit account details, and manage their preferences. This included creating a dashboard for users to track past orders and status.

Day 4: Product Review System

Date: Thursday, June 20, 2024

Added a review system for products. Customers can leave ratings and reviews for products they have purchased. Implemented a system for displaying the average rating on product pages.

Day 5: Order Management System (Admin Panel)

Date: Friday, June 21, 2024

Built an order management system for admins to view, manage, and update orders. Admins can mark orders as processed, shipped, or delivered, and send notifications to users regarding the status of their orders.

Day 6: Enhanced UI and User Experience

Date: Saturday, June 22, 2024

Improved the user interface (UI) by refining the design and adding more interactive elements. Implemented features like product search, filters, and enhanced cart views for a better user experience.
